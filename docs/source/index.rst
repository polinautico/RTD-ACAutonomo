Ahoy! Bem vindo a documentação sobre o projeto **AC Autônomo**, maraujos!
Nesta guia, iremos mostrar todo o projeto de criação da nossa embarcação não-tripulada: o AC Autônomo.

A ideia
-----

A partir do interesse dos membros da área de Elétrica e Computação do grupo em desenvolver um projeto maior, embarcamos na jornada rumo a participar da `Roboboat <https://roboboat.org>`_, uma competição focada em barcos autônomos.

A embarcação
-----

Nosso primeiro objetivo é autonomizar um barco, e por isso escolhemos o Álvaro de Campos, embarcação que estava parada em nossa oficina e foi utilizado pela equipe em competições no passado.

Utilizando um barco pronto não atrapalhamos o fluxo de trabalho dos membros da área de Casco e Propulsão e temos uma base sólida de um barco funcional para trabalhar.

Objetivo
-----

Nosso objetivo neste barco é construir um protótipo para entender os fundamentos da navegação autônoma. Este primeiro desenvolvimento servirá para acumular conhecimentos base sobre os componentes eletrônicos utilizados, como o GPS, o espaço necessário para abrigá-los e qual a influência do formato do casco no desempenho final, para que em um futuro projeto mais dedicado possamos ter um ponto de partida mais definido.
